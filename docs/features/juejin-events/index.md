# 掘金活动

`<Carousel>`元素里面可以放任意元素，比如放上两张图片就是掘金活动的效果。

:::demo

```vue
<template>
  <Carousel style="width: 470px; height: 280px;">
    <img
      src="/assets/juejin1.png"
      style="height: 280px;"
    />
    <img
      src="/assets/juejin2.png"
      style="height: 280px;"
    />
  </Carousel>
</template>
```

:::
